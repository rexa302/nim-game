# 🔥 GenLayer Nim Game

A simple browser-based Nim Game built with HTML, CSS, and JavaScript.

Players take turns removing matchsticks from the board.  
The player who removes the last matchstick wins the game.

---

# 🎮 How to Play

- The game starts with 4 rows of matchsticks:
  
1, 3, 5, and 7

- On your turn:
  - Select one or more matchsticks
  - You can only select matchsticks from ONE row
  - Press the **TAKE MATCHES** button

- Then the AI makes its move.

- The player who takes the final matchstick wins.

---

- Interactive matchstick animations
- Smart AI opponent using Nim strategy
- Responsive design for desktop and mobile
- Restart and New Game buttons
